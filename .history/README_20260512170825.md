# Google Maps Location Directions App

A Next.js web application that allows you to view locations from a CSV file on Google Maps and get directions to them from your current location.

## Features

- 📍 **Current Location Detection**: Get your current location using browser geolocation
- 🗺️ **Google Maps Integration**: View all locations on an interactive map
- 📋 **CSV Support**: Upload CSV files with location data
- 🧭 **Directions**: Click any location to see driving directions and distance/duration
- 📱 **Responsive Design**: Works on desktop and mobile devices
- 📲 **PWA Support**: Mobile manifest for app-like experience on mobile devices
- 🎨 **Tailwind CSS**: Beautiful, responsive UI

## Prerequisites

- Node.js 18+ 
- Google Maps API Key (get one from [Google Cloud Console](https://console.cloud.google.com/))

## Installation

1. **Clone or navigate to the project directory**
   ```bash
   cd gmap
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up Google Maps API Key**
   - Go to [Google Cloud Console](https://console.cloud.google.com/)
   - Create a new project
   - Enable the Maps JavaScript API
   - Enable the Directions API
   - Create an API key
   - Copy the API key and update `.env.local`:
     ```bash
     NEXT_PUBLIC_GOOGLE_MAPS_API_KEY=your_api_key_here
     ```

## CSV File Format

Your CSV file should have the following columns:

```
FID,Name,Latitude,Longitude
0,6,25.53288,82.21954
1,7,25.49883,82.3259
...
```

- **FID**: Unique identifier
- **Name**: Location name/number
- **Latitude**: Latitude coordinate
- **Longitude**: Longitude coordinate

## How to Use the App

1. **Browser Permissions**: Allow the browser to access your current location when prompted
2. **View Default Locations**: The app loads a default `locations.csv` from the public folder
3. **Upload Custom CSV**: Click the "Upload CSV" button to load your own location data
4. **Select a Location**: Click on any location in the sidebar to select it
5. **View Directions**: The map will show:
   - Your location (blue marker)
   - All available locations (yellow markers)
   - Selected location (red marker)
   - Directions route from your location to the selected location
   - Distance and duration information

## Troubleshooting

### Geolocation not working
- Ensure HTTPS is used (required for geolocation)
- Check browser permissions for location access
- Some devices/browsers may have restrictions on geolocation

### Map not displaying
- Verify your Google Maps API key is correctly set
- Check that Maps JavaScript API and Directions API are enabled
- Check browser console for error messages

### CSV file not loading
- Verify CSV format matches the required columns
- Check file encoding is UTF-8
- Ensure latitude/longitude are valid numbers
